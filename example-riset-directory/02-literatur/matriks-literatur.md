# Matriks Studi Literatur (Literature Review)

Berikut adalah ringkasan matriks dari beberapa jurnal dan referensi literatur yang saya gunakan sebagai landasan dalam menyusun riset perbandingan platform web dan mobile untuk pembelajaran Mata Kuliah Sistem Operasi:

| Judul / Penulis (Tahun) | Fokus Studi | Temuan Utama | Relevansi untuk Riset Saya |
|------------------------|-------------|--------------|---------------------------|
| Alfita dkk. (2024) — Analisis Keefektifan Penggunaan Aplikasi Berbasis Web dan Mobile pada Matkul Sistem Operasi UNNES | Evaluasi perseptual platform (Elena, YouTube, E-book) pada 50 mahasiswa Sistem Operasi UNNES | Elena 79.4%, YouTube 78.3%, E-Book 75% keefektifan. Evaluasi hanya perseptual, tidak membedakan akses web vs mobile secara eksplisit | Menjadi landasan konteks utama riset saya — menunjukkan bahwa mahasiswa Sistem Operasi UNNES memang menggunakan lebih dari satu platform, namun belum ada pengukuran teknis objektif |
| Dalimunthe dkk. (2025) — Analisis Perbandingan Efisiensi E-Learning Berbasis Website dan Aplikasi Mobile | Survei 90 mahasiswa, 9 indikator persepsi, t-test | Sig > 0.05 semua indikator — tidak ada perbedaan signifikan antara platform web dan mobile | Memperkuat urgensi gap penelitian saya: penelitian ini tidak mengukur performa teknis secara objektif dan tidak terikat konteks mata kuliah spesifik |
| Ridho dkk. (2018) — Perbandingan Performa PWA dan Mobile Web | Pengujian laboratoris teknis: response time, memory, storage | PWA unggul untuk file besar; waktu respons dipengaruhi ukuran file dan frekuensi akses | Memberikan dasar metodologis pengukuran teknis yang saya adopsi dengan Google Lighthouse — membuktikan bahwa performa teknis bisa berbeda signifikan antar platform |
| Prasetyaningsih & Muchtar (2023) — Perbandingan UX Website dan Mobile Shopee via UEQ | UEQ + FGD, 30 responden | Platform web dan mobile masing-masing unggul pada dimensi yang berbeda | Memberikan bukti bahwa perbandingan per dimensi (bukan hanya overall) penting — menjadi justifikasi penggunaan CSUQ yang memiliki 3 subdimensi spesifik |
| Wahyudi (2022) — SLR Pengembangan Aplikasi Web dan Android sebagai Penunjang Kerja | Systematic Literature Review, 15 artikel 2015–2022 | Sektor pendidikan dominan (6 artikel), mayoritas fokus rancang bangun bukan perbandingan | Menunjukkan bahwa riset perbandingan web vs mobile masih jarang dilakukan secara terintegrasi — memperkuat posisi gap penelitian saya |
