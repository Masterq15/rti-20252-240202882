# Rencana Penelitian

## Ringkasan

| Item | Keterangan |
|------|-----------|
| Judul | Perbandingan Kepuasan Pengguna dan Performa Teknis Platform Web dan Mobile pada Pembelajaran Mata Kuliah Sistem Operasi |
| Target Publikasi | Jurnal nasional terakreditasi SINTA |
| Instrumen Kepuasan | SUS (System Usability Scale) + CSUQ (Computer System Usability Questionnaire) |
| Instrumen Teknis | Google Lighthouse (FCP, TTI, TBT, Speed Index) |
| Desain | Kuantitatif komparatif, between-subject, ≥ 60 mahasiswa |
| Masalah | Mahasiswa Sistem Operasi menggunakan platform web (Elena UNNES) dan mobile secara bersamaan, namun belum ada data empiris yang membandingkan keduanya dari sisi kepuasan dan performa teknis |
| Solusi | Pengukuran terintegrasi: kuesioner SUS + CSUQ + audit Lighthouse |
